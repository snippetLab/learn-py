## 🧠 Overview  
**PythonLab** is my personal space to explore, understand, and practice the **fundamentals of Python programming**.  
This repository contains simple, structured examples for each topic — focusing on clarity, logic, and consistency.  

---

## 📚 Topics Covered  
| Category | Description |
|-----------|-------------|
| 🐍 **Basics** | Variables, data types, operators, input/output |
| 🔁 **Conditional & Loops** | If-else, nested conditions, for and while loops |
| 🧩 **Functions** | Defining functions, arguments, return values, lambda |
| 📦 **Data Structures** | Lists, tuples, sets, dictionaries, string operations |
| 🧱 **OOP (Object-Oriented Programming)** | Classes, objects, constructors, inheritance, polymorphism |
| ⚙️ **Modules** | Importing, creating and using custom modules |
| 📂 **File Handling** | Reading, writing, and working with text files |
